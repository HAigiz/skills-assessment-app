# SkillExam - Платформа оценки навыков IT-команды

## Описание проекта

SkillExam - веб-приложение для учета и визуализации навыков сотрудников IT-компании. Система позволяет сотрудникам проводить самооценку, руководителям подтверждать или корректировать оценки, а HR видеть сводную картину по отделам и планировать развитие.

## Функционал

### Для сотрудника:
- Авторизация по логину/паролю
- Просмотр списка навыков по категориям
- Самооценка навыков по шкале 1-5
- Просмотр профиля компетенций (радарная диаграмма)
- История изменений оценок

### Для руководителя:
- Просмотр профилей сотрудников своего отдела
- Подтверждение или корректировка оценок сотрудника
- Сравнение навыков нескольких сотрудников
- Поиск сотрудников по навыку

### Для HR/администратора:
- Управление справочником навыков
- Сводный отчет по отделам
- Экспорт данных в CSV
- Аналитика и статистика

## Технологический стек

- **Backend:** Python + Flask
- **База данных:** PostgreSQL
- **Frontend:** HTML + CSS + JavaScript + Bootstrap + Chart.js
- **Контейнеризация:** Docker + Docker Compose
- **Аутентификация:** Flask-Login

## Структура БД

5 таблиц:
1. `users` - пользователи
2. `departments` - отделы
3. `skills` - справочник навыков
4. `skill_assessments` - оценки навыков
5. `assessment_history` - история изменений оценок
